Project overview:
The project automates end to end workflow by generator creation, billing configuration, Service setup ,route assignment, price book update, and logout functionality in the application.
The automation validates important fields and verifies that the workflow is completed successfully using browser automation.
Automation tool:
Selenium WebDriver 4-->Used for automating browser actions such as clicking elements, entering data, navigating pages, and validating application behavior.
Programming Language:
Java-->Used to develop automation test scripts and implement logic for test execution.
IDE:
Eclipse IDE-->Used for writing , debugging, and executing automation script.
Browser:
Updated version of Google Chrome-->Used for executing automation test cases.
Driver:
ChromeDriver-->Used to establish communication between Selenium WebDriver and Chrome browser
Version Control:
Git-->Used for version management and tracking code changes.
